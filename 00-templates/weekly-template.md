## 二、知识点详解

### 2.1 PyCharm Git 操作

**核心概念：**
> PyCharm 内置了 Git 功能，不用记命令行，点按钮就能上传

**详细笔记：**
- 用 **Commit** 按钮保存本地记录
- 用 **Push** 按钮上传到 GitHub
- 绿色文件 = 新增，蓝色文件 = 修改

**遇到的问题 & 解决：**
- ❓ 问题：找不到 Commit 按钮
- ✅ 解决：在顶部菜单栏 **Git** → **Commit...**，或按 `Ctrl + K`
