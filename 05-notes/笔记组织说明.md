## 📋 操作步骤

### 新建一讲笔记
1. 打开 PyCharm，进入 `学习笔记/` 目录
2. 找到对应课程文件夹（没有就新建）
3. 右键 → New → File → 命名为 `第XX讲-关键词.md`
4. 打开 `笔记模板_Python学习.md`，复制模板内容到新文件
5. 边看课边填

### 快速记录
- 不想用完整模板时，复制 `快速笔记模板.md` 的内容

### 新课程开始前
- 在 `学习笔记/` 下新建课程文件夹，如 `景霄-Python核心技术36讲/`
- 按课程模块建子文件夹
- 子文件夹内按讲次命名文件

## 💡 PyCharm 使用技巧

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| Markdown 预览 | Ctrl+Shift+I（macOS：Cmd+Shift+I） | 左右分栏预览 |
| 代码块内代码补全 | 在 ```python 块内直接输入 | PyCharm 支持 Markdown 内代码补全 |
| 格式化表格 | 安装 Markdown 插件 | 推荐「Markdown Navigator Enhanced」 |
| 目录导航 | Ctrl+B / Cmd+B | 点击标题跳转 |
| 搜索所有笔记 | Ctrl+Shift+F | 全局搜索关键词 |

## ⚠️ 注意事项

- **文件名只用中文+数字+短横线**，不用空格和特殊字符
- **代码块必须标注语言**：```python，不要写 ```
- **不存截图**到笔记文件夹，图片会让 .md 文件难以管理
- **每周备份一次**笔记文件夹到 GitHub 或云盘
