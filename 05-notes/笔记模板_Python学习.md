# 课程精通学习笔记模板

> **用途**：在 PyCharm 中每学完一讲，复制此模板新建笔记文件
> **文件命名**：`第XX讲-标题关键词.md`，例如 `第15讲-列表推导式.md`
> **存放路径**：在课程文件夹下按模块建子文件夹，例如 `尹会生/高级数据类型/`

---

## 使用说明

1. **复制此模板** → 粘贴到 PyCharm 中新建 `.md` 文件
2. **改文件名** → `第XX讲-核心关键词.md`
3. **边学边填** → 不是学完再记，是学的时候同步填
4. **留白原则** → 不知道就写「？」，学完后回头补
5. **代码必须手敲** → 绝不复制粘贴讲师代码到笔记里
